bitwise operator

bitwise AND: & - if both LHS and RHS 1 = 1

bitwise OR : | - if both LHS and RHS 0 = 0

Exclusive OR: ^ - if both LHS and RHS are same = 0, else = 1

Bitwise complement(urnary) : ~ - 0 =>1， 1=>0， positive => add 1 and then change the sign negative => sub 1 and then change the sign
